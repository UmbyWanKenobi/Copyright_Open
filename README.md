# Copyright_Open
Certifica automaticamente post/pagine su OpenTimestamps (Bitcoin). Aggiunge badge con certificato .ots, istruzioni di verifica e supporta cronologia o sovrascrittura dei certificati.
WP Bitcoin Timestamp è un plugin WordPress che integra il protocollo **OpenTimestamps** per creare prove crittografiche di esistenza dei tuoi contenuti direttamente sulla blockchain di Bitcoin – **gratuitamente, in modo decentralizzato e senza intermediari**.

Ideale per:
- Ricercatori e scienziati che vogliono dimostrare priorità su idee o previsioni
- Giornalisti e blogger che necessitano di protezione da plagio
- Creator che vogliono certificare la data di pubblicazione di articoli, note o scoperte

## Come funziona

1. Il plugin calcola l'hash SHA-256 del contenuto dell'articolo (o di parti selezionate).
2. Invia l'hash a uno o più calendar server OpenTimestamps pubblici.
3. Riceve una prova Merkle che aggrega il tuo hash con migliaia di altri in una singola transazione Bitcoin.
4. Salva la prova (.ots) e mostra un certificato verificabile direttamente nell'articolo o in backend.

La verifica è possibile in qualsiasi momento con il client ufficiale OpenTimestamps – anche fra decenni.

## Funzionalità principali

- Timestamp automatico alla pubblicazione o manuale
