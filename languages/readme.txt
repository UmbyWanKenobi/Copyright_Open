Traduzioni per Copyright_Open

Per tradurre il plugin:

1. Scarica Poedit (https://poedit.net/)
2. Apri il file copyright-open.pot
3. Crea una nuova traduzione per la tua lingua (es: it_IT)
4. Salva come copyright-open-it_IT.po
5. Compila per generare il .mo
6. Inserisci i file nella cartella languages:

/copyright-open/
  languages/
    copyright-open.pot
    copyright-open-it_IT.po
    copyright-open-it_IT.mo

Lingue supportate:
- Italiano (it_IT) - Incluso
- Inglese (en_US) - Default
- Aggiungi la tua...

Per contribuire con una traduzione:
1. Completa il file .po per la tua lingua
2. Invia una Pull Request
3. O invia il file a support@esempio.com
